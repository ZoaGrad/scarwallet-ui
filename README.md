## Quickstart

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ZoaGrad/scarwallet-ui.git
    cd scarwallet-ui
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm start
    ```

    Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
